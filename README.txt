功能
将csv格式文档的每一行以vector<string>方式输出
1、可以指定列的qutoa，列分割符，行分割符，缺省值分别为"，,和\n
2、有qutoa的列可以包含\",,和\n，可以有任意个
3、无qutoa的列可以包含\,，\"和\\n，可以有任意个
4、可以有任意多空行
5、可以空列，少列
6、转换后列中带\n都被替换为空格

算法：
用两块缓冲长度缓存
用二进制读，不进行任何转义
当行长小于块长度时！！！用两块缓冲长度保证一行数据是完整的。
当行跨到第二块时，移动第二块数据到第一块，再读入第二块数据
当行读到\0时表示最后一行
当行跨过第二块时，行长已超过块长度，报错
